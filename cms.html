How GitHub Pages Custom Domains Work

To point a custom domain to your GitHub Pages site:

  1. Add your domain in the field above — this creates a CNAME file in your repo
  2. In your domain registrar (e.g. GoDaddy, Namecheap), add these DNS records:
TypeNameValue
A@185.199.108.153
A@185.199.109.153
A@185.199.110.153
A@185.199.111.153

DNS propagation typically takes 24–48 hours. GitHub Pages will automatically provision an SSL certificate via Let's Encrypt.
