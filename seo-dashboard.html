Bing Webmaster Tools — API Migration Required

Microsoft deprecated API key authentication for the Bing Webmaster Tools API in 2024. Access now requires Microsoft OAuth (Azure AD). The old API key is no longer accepted.

To restore Bing data, you need to:

  1. Go to portal.azure.com → App Registrations
  2. Create a new app → Single tenant → copy the Client ID
  3. Add a Client Secret → copy the Client Secret value
  4. In Bing Webmaster Tools → Settings → API Access → grant your app access
  5. Save Client ID + Secret into /Users/steven.hill/nanoclaw/secrets/bing-oauth.json

Once set up, the dashboard will auto-use OAuth tokens and this notice will disappear.
